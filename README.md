# product360
Product 360 viewer



Example:
```
  <div id="slider"></div>
  <script src="https://cdn.zipview.kr/public/javascript/product360.js"></script>
  <script>
    const pdSlider = new Product360('#slider', {
      src: 'https://cdn.zipview.kr/webvr/service/DEMO/estate/ISO/iso001_min.jpg',
      sensitivity: 2,
    });
  </script>
```
